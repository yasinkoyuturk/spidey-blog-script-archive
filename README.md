# Spidey Blog Script v1.5 - Historical Archive

A historical archive of classic ASP web scripts developed by Yasin "Spidey" Koyutürk between 2006-2007.

## About This Project
This repository contains the original source code for the "Spidey Blog Script v1.5". During the early 2000s, this ASP-based blog management system was widely distributed and highly rated across major Turkish webmaster platforms, including ASPindir and ASPRehberi.

It was originally deployed on the legendary Turkish gaming portal infrastructure, oyuncum.net.

**Note:** This code is strictly uploaded for digital archiving and internet archaeology purposes. It is not maintained for modern server environments.
